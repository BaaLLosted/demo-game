## «Анатомия скама: Изнанка»

«Анатомия скама: Изнанка»

Информация об авторе: Бальцевич Александр Викторович, Андрушкевич Илья Андреевич, Кибербезопасность, СДП-КБ-241

Учебная игра по теме социальной инженерии и методов защиты от мошенничества.

## Технологии
- HTML
- CSS
- JavaScript
- Node.js
## 🛠 Требования
- Node.js 14+
## 🚀 Как запустить
1. Клонируйте репозиторий:
git clone https://github.com/BaaLLosted/demo-game.git
cd repo
2. Установите зависимости:
npm install
3. Запустите проект:
npm start
