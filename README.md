# Анатомия скама: Изнанка

Интерактивный симулятор социальной инженерии.

## Особенности реализации
- **Скрытый таймер**: 0-20 сек (без штрафа), 20-40 сек (-7%), 40+ сек (-20%).
- **Логическая проверка**: Жертвы дают прямые указания (например, "не присылать ZIP"). Нарушение этих условий ведет к провалу.
- **Конструктор**: Игрок собирает каждое сообщение из Триггера, Стиля и Инструмента.
- **Динамика**: Имитация ожидания ответа ("Объект печатает...").

## Технологии
- HTML5, CSS3, Vanilla JavaScript.

## Запуск
Откройте `index.html` в браузере.
